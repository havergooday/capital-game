# 🌍 오늘의 수도 · 오늘의 나라

방향 없이 **거리만**으로 오늘의 정답을 맞히는 일일 지리 퀴즈 게임.  
[Wordle](https://www.nytimes.com/games/wordle/index.html) 형식에서 영감을 받아, 세계 지리 지식을 테스트합니다.

---

## 게임 방법

1. **수도 모드** 또는 **나라 모드** 선택
2. 수도/국가 이름을 입력하고 제출
3. 정답과의 거리(km)가 색상 막대와 함께 표시됨
4. 거리를 단서로 추측 범위를 좁혀 정답을 맞힘
5. 매일 자정 새로운 문제 출제 (전 세계 동일 정답)

> **방향 힌트 없음** — Worldle보다 훨씬 도전적입니다.

---

## 거리별 색상 기준

| 거리 | 색상 |
|------|------|
| 0 km (정답) | 🟢 초록 |
| 500 km 미만 | 🟢 진초록 |
| 1,500 km 미만 | 🟡 라임 |
| 3,000 km 미만 | 🟡 노랑 |
| 6,000 km 미만 | 🟠 주황 |
| 6,000 km 이상 | 🔴 빨강 |

---

## 기술 스택

- **순수 HTML / CSS / Vanilla JS** — 빌드 도구 없음
- **단일 파일** (`index.html`) — 모든 코드 포함
- **외부 의존성**: Wikipedia REST API (정답 공개 후 정보 표시)
- **배포**: GitHub Pages (정적 호스팅)

---

## 주요 기능

- 세계 195개 수도 / 195개 국가 데이터 (한국어)
- 자동완성 입력 지원
- 추측 결과 거리순 정렬
- 정답 맞힌 후 Wikipedia 썸네일 · 설명 자동 로드
- 결과 텍스트 클립보드 공유
- URL 해시 라우팅 (`#capital`, `#country`) — 직접 링크 공유 가능
- 다크모드 전용, 모바일 최적화 (max-width 480px)
